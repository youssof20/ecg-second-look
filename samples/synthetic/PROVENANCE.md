# Synthetic sample provenance

All files in this directory are generated by `scripts/generate_synthetic_ecg.py`.

- No patient data
- No hospital identifiers
- Banner text on the clean page is explicit fiction: `SYNTHETIC ECG FIXTURE — NO PATIENT DATA`

Distortions (`photo_blurry`, `photo_glare`, `photo_shadow`, `photo_skewed`, `photo_tiny`) are algorithmic overlays for regression tests, not clinical photographs.
